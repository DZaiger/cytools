A software package for analyzing Calabi-Yau hypersurfaces in toric varieties.

Get Started

Fast algorithms

CYTools includes fast algorithms to handle and triangulate 4D reflexive polytopes and obtain the data of the associated Calabi-Yau hypersurfaces.

Powerful and robust

CYTools can be used to explore the entirety of the Kreuzer-Skarke database, even at the largest Hodge numbers.

Easy to use

CYTools is easy to install and use. We provide extensive documentation and all the code is open-source.
